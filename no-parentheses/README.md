# No Parentheses

Programming challenges, but no parentheses are allowed.

## Challenge rules

1. The characters `(` (`U+0028 LEFT PARENTHESIS`) and `)` (`U+0029 RIGHT PARENTHESIS`) or escapes of the same must not appear in the challenge code.
2. The challenge code must not derive a parenthesis character.
